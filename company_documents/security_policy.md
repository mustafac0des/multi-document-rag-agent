# Aether Corp Security Policy
_INTERNAL_

---

## 1. Data Classification

| Level        | Description             |
|--------------|-------------------------|
| Public       | Safe to share           |
| Internal     | Company use only        |
| Confidential | Sensitive business data |
| Restricted   | Critical/exec-only      |

---

## 2. Access Control

- Role-Based Access Control (RBAC)
- Least privilege enforced

---

## 3. Authentication

- 2FA mandatory
- Password rotation every 90 days

---

## 4. Device Security

- Disk encryption required
- No unauthorized software
- Automatic updates enforced

---

## 5. Monitoring

- Continuous logging
- Anomaly detection alerts

---

## 6. Incident Response

1. Detect  
2. Contain  
3. Investigate  
4. Resolve  
5. Report  

---

## 7. Violations

Consequences:
- Access revocation
- Termination
- Legal escalation
